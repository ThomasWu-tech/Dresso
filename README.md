# Dresso - AI Fashion Assistant

Dresso is a modern, AI-powered fashion assistant designed to help users organize their wardrobe and discover new outfit combinations. By leveraging AI, Dresso provides personalized style recommendations based on the items you already own.

- **Virtual Closet**: Keep track of all your clothing items and saved coordinations in one place.
- **Outfit Creator**: Mix and match different items from your closet to visualize new looks.
- **AI-Generated Looks**: Get personalized outfit suggestions generated by AI based on your wardrobe.
- **Secure Authentication**: Robust signup and login system using email and password.
- **User Profiles**: Manage your personal information and style preferences.
- **Light Theme**: Clean and modern user interface optimized for clarity and ease of use.

## âš™ï¸ Setup Instructions

### 1. Backend Setup
Navigate to the `backend` directory and install the required Python packages:

```bash
cd backend
pip install -r requirements.txt
```

Start the FastAPI server:
```bash
uvicorn main:app --reload
```
The backend will be available at `http://127.0.0.1:8000`.

### 2. Frontend Setup
The frontend uses CDN-hosted scripts for React and Tailwind CSS, so no complex build step is required. Simply open the HTML files in your browser.

For local development, you can use a simple live server (like the VS Code Live Server extension) to serve the root directory.
